# Product Overview

仕事サポートアプリ（Work Support）は、日常の業務を効率化する Web アプリケーションです。個人の工数管理、目標達成の追跡、週報作成を統合的に支援し、業務の可視化と生産性向上を実現します。

## Core Capabilities

1. **工数管理**: 案件・タスク単位で見積と実績を登録し、工数データを一元管理
2. **目標管理**: 期間ベースの目標設定と進捗トラッキング
3. **週報出力**: 週次の工数サマリー・ミッション・目標進捗を整形して表示・出力
4. **認証・プロフィール**: Supabase 認証基盤によるセキュアなユーザー管理

## Target Use Cases

- 日々の工数を記録し、チームへの報告や自己管理に活用
- 期間目標を設定し、達成状況を振り返る
- 週報を自動生成し、定型業務の時間を削減

## Value Proposition

- **仕様駆動開発**: `@docs/specs` に仕様を集約し、設計と実装の一貫性を担保
- **型安全な共有スキーマ**: `packages/shared` で API コントラクトを共有し、フロントエンド・バックエンド間の型整合性を維持
- **モダンなモノレポ構成**: pnpm ワークスペースによる効率的なパッケージ管理
